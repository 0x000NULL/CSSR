Groups

The Group tab contains a list of the current computer's groups. An administrator can assign each user or group to various groups.

To add a user or group to score on, choose a user or group from the dropdown list under the Users heading and click the Add button to append them to the Added Users list.

To remove a user or group from the Added Users, select the user and click the Remove button.

Select the Scoring checkbox to have the Scoring Engine score on the specified list of users and groups.
