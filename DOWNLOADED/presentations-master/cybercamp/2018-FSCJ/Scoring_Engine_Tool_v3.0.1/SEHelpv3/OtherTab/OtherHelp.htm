Other Tab

In the Other tab, specify various scoring criteria, such as checking the host file configuration.

Allow or disallow Remote Desktop connections.

You can also create a list of files to be removed by the user.
To add a file, click the Add File button and navigate to the specified file to be removed via Windows Explorer.
To remove a file, select the entry in the File Path box and click the Remove Selected File button.

To score on these items, select the Scoring checkbox next to each item.

Scoring is performed when the user configures Remote Desktop settings, verifies the host file and removes selected items accordingly.
