Features Tab

The Features tab contains a list of detected Windows features such as Games, Media Features, .NET Framework, etc.

Each item can be set to Enabled or Disabled.

Select the desired value for each item and select the Scoring checkbox to score on that item.

Scoring will be performed when the user configures the features to match the specified values in the list.

Note that the Scoring Engine takes about five seconds to check each item that has been set to scoring, therefore selecting multiple features will require longer update time.
