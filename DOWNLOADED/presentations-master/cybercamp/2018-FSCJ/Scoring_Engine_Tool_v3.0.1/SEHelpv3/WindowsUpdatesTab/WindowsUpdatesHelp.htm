Windows Updates

In the Windows Update tab, select a type of update to score on, for example, Install updates automatically and select the Scoring checkbox above.

In the Windows Update Title, insert the description of a new Windows update.

In the Windows Update Id, insert a KB value of the new Windows update. The scoring will be performed on a valid KB value.

To remove an entry from the Windows Update List, select the desired entry and click Remove Selected Entry.

In the Windows Update List, select the Scoring checkbox to score on the specified entry.
